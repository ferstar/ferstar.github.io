大概罗列下常用的东西, 备忘性质, 并不会写具体怎么怎么用. 二八定律, 这些工具功能的百分之二十足以支撑平时百分之八十甚至更多的使用需求

字体

  1. YaHei Consolas Hybrid - 雅黑 Consolas合体
  2. SourceCodePro
  3. MacType - 全局字体显示效果微调 后来发现这东西其实并没有什么卵用, 屏幕垃圾的话, 怎么调字体都是虚的, 换显示器才是王道

编辑器

VIM

对这玩意一直心存敬畏,一般只在服务器改参数时用, 或者实在环境恶劣, 只有一个terminal的情形下使用, 所幸这种环境不常遇到, 当然遇到了也不虚

Notepad++

常用配置可以利用"设置->首选项->云端"把路径设置到dropbox自定目录下面, 保持配置同步, 另外需要把白底配色换掉, 一直看白底黑字眼睛要爆炸 作为一个python党, 一定要注意两个设置, 一个是把tab键用4个空格代替, 一个是编码UTF-8

Sublime Text 3

本来是想用Atom来着,不过那货实在是太慢, 所以还是用这个啦 插件的话, 装完这个爸爸辈的 Package Control 其他随便 install, 很方便
